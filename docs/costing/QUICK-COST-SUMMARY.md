## ❓ FAQ

### Q: Can I really launch for $15/month?
**A:** Yes! Free hosting tiers + 2 small servers = $14-15/month total.

### Q: What if I get 1,000 users on day 1?
**A:** Free tiers can handle it! Vercel serves 100GB bandwidth free. You might need to upgrade servers to $25 each ($50 total).

### Q: How much does Gemini API cost at scale?
**A:** $0.0005 per analysis. 10,000 analyses = $5/month. Super cheap!

### Q: What if GitHub rate limits me?
**A:** Use authenticated API (5,000 req/hour FREE). That's 120K requests/day. More than enough.

### Q: Do I need to pay for SSL?
**A:** No! All modern hosts (Vercel, Render, Netlify) include free SSL.

### Q: What about backups?
**A:** MongoDB Atlas includes automatic backups in all paid tiers (M2+). Free tier doesn't have backups, but you can manually export.

### Q: Can I use AWS instead?
**A:** Yes, but it's more expensive and complex. Render/Vercel are better for startups.
