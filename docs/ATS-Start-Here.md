# ATSMaster: Project Overview - Everything You Need to Know

---

## 📦 WHAT YOU'RE BUILDING

**ATSMaster** is a complete AI-powered recruitment intelligence platform that:

1. **Analyzes Resumes** (Phase 1) - Scores compatibility with job requirements
2. **Powers Recruiters** (Phase 2) - Provides tools for bulk screening and insights
3. **Verifies Credentials** (Phase 3) - Autonomously validates candidate claims with AI agents

**In 3 words:** Resume AI + Verification Agents
